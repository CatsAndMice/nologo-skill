# nologo-open-api

无水印解析API调用工具

## 快速开始

```bash
cd nologo-open-api/references

# 设置Token (自动保存到config.json)
node api-client.js set-token <your-token>

# 解析链接
node api-client.js parse "https://v.douyin.com/xxxxx"
```
